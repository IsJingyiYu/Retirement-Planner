# Product Overview

## Product Position

Retirement Planner is designed for people entering retirement or adjusting to early retirement life.

The product does not optimize the user's entire life.
It helps with one immediate problem:

**"What can I start with today?"**

## Core Value

The Mini Program acts as a daily life starter.
It reduces hesitation, lowers the emotional cost of getting started, and helps users rebuild rhythm through one small action each day.

## MVP Features

### 1. Onboarding Survey

Collect only a few cold-start signals:

- interests
- current city
- activity preferences
- social tendency

### 2. Daily Anchor

Each day the user gets one suggested starting point, such as:

- go downstairs for a short walk
- send a message to one friend
- spend 10 minutes learning something new

User actions:

- completed
- swap for another option

### 3. Local Suggestions

Use city, weather, and interest tags to recommend:

- parks
- exhibitions
- community activities

### 4. Next-day Reminder

The reminder is opt-in only.
The user explicitly chooses whether to be reminded the next day.

## MVP Success Signals

- percentage of users who finish onboarding
- daily anchor completion rate
- swap rate before completion
- revisit rate on day 2 and day 7
- reminder opt-in rate
