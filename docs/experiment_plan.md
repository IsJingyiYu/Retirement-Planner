# Experiment Plan

## Goal

Validate whether a daily anchor can improve early habit formation and short-term retention for newly retired users.

## Hypotheses

### H1

Users are more likely to return if they receive exactly one daily starting suggestion instead of a list of tasks.

### H2

Local suggestions improve perceived usefulness because they make the product feel practical and real.

### H3

Optional reminders perform better than default push behavior because the product feels supportive instead of demanding.

## MVP Experiments

### Experiment A: Anchor Framing

- version A: "Today's start"
- version B: "One thing you can do today"

Measure:

- completion rate
- swap rate

### Experiment B: Local Suggestions Placement

- version A: below the daily anchor
- version B: on a separate page

Measure:

- click-through rate
- same-day return rate

### Experiment C: Reminder Timing

- version A: ask after completion
- version B: ask after the user opens tomorrow preview

Measure:

- opt-in rate
- next-day open rate

## Metrics To Track

- onboarding completion
- anchor exposure
- anchor completion
- anchor swap
- local recommendation click
- reminder opt-in
- D2 retention
- D7 retention
